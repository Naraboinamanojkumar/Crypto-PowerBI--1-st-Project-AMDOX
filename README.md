# AMDOX Cryptocurrency Market Analysis Dashboard

## Project Overview

This project was developed as part of an analytics project at AMDOX.
The objective is to analyze historical cryptocurrency market data and visualize key trends using Microsoft Power BI.

## Tools and Technologies

* Microsoft Power BI
* Data Visualization
* CSV Dataset
* Cryptocurrency Market Data

## Dataset

The dataset contains historical price data for multiple cryptocurrencies including:

* Bitcoin
* Ethereum
* Dogecoin
* Cardano
* Solana

## Dashboard Features

The Power BI dashboard provides:

* Cryptocurrency price trend analysis
* Market comparison between different cryptocurrencies
* Historical performance visualization
* Interactive charts and filters

## Project Structure

dashboard/

* Amdox_powerbi_team5.pbix → Power BI dashboard file

dataset/

* Cryptocurrency Historical Prices.zip → Dataset used for analysis
